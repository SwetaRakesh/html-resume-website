# HTML Resume Website

A personal resume website built using pure HTML5, focusing on strong fundamentals, semantic structure, and accessibility.


## Features

- Semantic HTML5 structure (`header`, `nav`, `section`, `article`, `footer`)
- Clean and readable content hierarchy
- Internal page navigation
- HTML forms with proper labels (accessibility-friendly)
- Education displayed using tables
- Beginner-friendly and well-structured code
